# Bounce

A browser-based game.

## Setup & Development

This project is a static web application. 

1. **Local Development**: Open `index.html` in your browser.
2. **Deployment**: This project is configured to run on GitHub Pages.

## Deploying to GitHub Pages

1. Push your changes to the `main` branch.
2. Go to the repository settings on GitHub.
3. Scroll down to **Pages**.
4. Select `main` branch as the source.
5. Save. Your game will be available at the provided URL.
